About Smart Contact Manager

Smart Contact Manager is a secure and user-friendly web application designed to simplify contact management. Whether you're keeping track of friends, clients, or business connections, this tool helps you organize everything with ease.

  • Create and manage contacts effortlessly
  • Send emails directly from the dashboard
  • Login via Google, GitHub, or create your own account
  • Secure backend powered by Spring Boot and MySQL
